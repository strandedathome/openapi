# openapi

Run your own RPC node for [Rolls Wallet](https://pecanrolls.net/wallet/): Web Wallet for PecanRolls Blockchain

# Installation

```
git clone https://github.com/strandedathome/openapi.git

pip install -r requirements.txt

uvicorn openapi:app
```
